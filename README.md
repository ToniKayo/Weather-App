A simple Flask-based web application that fetches and displays real-time weather information using the OpenWeatherMap API.
![image](https://github.com/user-attachments/assets/1ab3be06-8a41-49fc-b1d2-d7fbb0446cff)

Features

Get current weather data for any city.

Displays temperature in Celsius and Fahrenheit.

Shows humidity, wind speed, and weather description.

Includes an icon representing the weather condition.

Handles errors gracefully (e.g., invalid city names, network issues).

Technologies Used

Python (Flask, Requests)

HTML, CSS (via Flask templates)

OpenWeatherMap API

Environment Variables

API_KEY: Your OpenWeatherMap API key. You can set it in the app.py file.

Usage

Enter a city name in the input field.

Click submit to get real-time weather information.

View temperature, weather description, humidity, wind speed, and an icon.

Error Handling

If an invalid city is entered, an error message will be displayed.

If there is a network issue, an appropriate error message will appear.
